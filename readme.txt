▼開発環境について

■タスクランナー
gulpを使用しています。
バージョンは下記になります。
npm : 8.19.2
node: v16.18.0

■フレームワーク
ejsを使用しています。

■ページの仕様について
ページ内のテキストは、/_src/json/内に格納している
data.jsonで管理しています。


・メインビジュアル
youtubeの画像をクリックするとlightboxとしてyoutubeが
開くようになっています。

・NEWSお知らせエリア
投稿記事の内容は/dist/json/内に格納している
news.jsonで管理しています。
news.jsonの投稿内容を増やすとページの投稿一覧の項目が増えます。
